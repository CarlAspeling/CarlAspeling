## 🚀 Projects

### [CounselWise](https://github.com/CarlAspeling/counsel-wise)
Full-stack platform for HPCSA-registered counsellors. Clinicians input patient intake data and receive evidence-based therapeutic intervention reports tailored to the individual client. Built around a weighted scoring recommendation engine, with automated report generation, SSO, and an analytics dashboard.

`Laravel` `Vue.js` `PostgreSQL` `Inertia.js` `Tailwind CSS`

### [Industrial Psychology Inventories](https://github.com/CarlAspeling/ind-psy-inv)
A personal project combining my psychology background with my development skills. A web app for building and administering psychometric inventories, built to explore Laravel, Filament, and Vue.js working together.

`Laravel` `Vue.js` `Filament` `MySQL`
